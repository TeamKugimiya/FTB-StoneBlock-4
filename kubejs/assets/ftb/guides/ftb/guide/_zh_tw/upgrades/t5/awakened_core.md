# 覺醒核心升級

<ItemImage id="draconicevolution:awakened_draconium_block" scale="3" />

本升級需要一座等級 8 的能量核心：也就是 378 塊<Color id="gold">覺醒龍金屬方塊</Color>和 786 塊<Color id="light_purple">龍金屬方塊</Color>。
