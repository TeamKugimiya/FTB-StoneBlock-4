# ME 光束成型器

本模組提供一套以「光束」傳輸的 AE2 設備：
- 方塊／部件版光束成型器：用於建立點對點連線。
- 全向光束成型器：用於建立多點連線。
- 無線能源塔：用於無線傳輸能量。
